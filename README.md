# DL-Simple-Examples

Simple Deep Learning Examples to use as templates and to compare performances.

## Datasets

MNIST dataset from : http://yann.lecun.com/exdb/mnist/ is being used for all examples.

## Dependencies

Pytorch     0.4.1\
torchvision 0.2.1\
matplotlib  2.2.2\
numpy       1.15.1
